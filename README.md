# QuantPulse — Financial Sentiment Terminal

A high-performance, real-time financial market sentiment analysis dashboard. This terminal uses simulated live data feeds and Vader-style NLP to analyze market mood, sentiment trends, and correlation with price action.

![QuantPulse Terminal](public/screenshot.png) *(Note: Add a real screenshot after deployment)*

## 🚀 Features

- **Live Sentiment Score**: Real-time gauge tracking market bullishness/bearishness.
- **Matrix Rain Background**: Cyberpunk-themed aesthetic with a custom canvas implementation.
- **3D Interactive Globe**: Built with Three.js to visualize global market hotspots.
- **Live News Feed**: Streaming news intelligence with automated sentiment tagging.
- **Sentiment vs Price Alignment**: Scatter plot correlation analysis using Chart.js.
- **Head-to-Head Battle**: Comparative analysis of top ticker sentiments (e.g., AAPL vs TSLA).
- **Fear & Greed Index**: Comprehensive market breadth visualization.

## 🛠️ Technology Stack

- **Structure**: HTML5
- **Styling**: Vanilla CSS3 (Custom Glassmorphism Design System)
- **Logic**: JavaScript (ES6+)
- **Graphics**: 
  - [Three.js](https://threejs.org/) (3D Globe)
  - [Chart.js](https://www.chartjs.org/) (Financial Charts)
  - [GSAP](https://greensock.com/gsap/) (Smooth UI Animations)
